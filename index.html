Antenna Array Factor

Uniform linear array beam pattern. Adjust element count, spacing, and scan angle to shape the main lobe.

AF(θ) = sin(N·ψ/2) / N·sin(ψ/2)

Smith Chart

Plot normalised load impedance on an interactive Smith Chart. Reads out Γ, VSWR, and return loss live.

Γ = (Z_L − Z₀) / (Z_L + Z₀)
