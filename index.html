RF Engineering Toolbox

Interactive calculators for RF & microwave systems analysis

Noah Hinckster  ·  EE Graduate Student  ·  GitHub  ·  Last updated: 2026-04-04
Batch 1 — Core RF Tools
📡
Tool 01

Friis Transmission

Calculate received power in a free-space link. Sweep distance or frequency and observe path loss in real time.

Pr = Pt + Gt + Gr + 20·log(λ/4πd)
🔊
Tool 02

Noise Figure Cascade

Friis cascade formula for up to 4 stages. See how each stage's NF and gain shape the system noise floor.

F = F₁ + (F₂-1)/G₁ + (F₃-1)/G₁G₂ + …
📈
Tool 03

IIP3 & Intermod

Third-order intercept and IM3 product calculator. Visualise the fundamental and spur intercept on a log–log plot.

P_IM3 = 3·Pin − 2·IIP3 + G
🔭
Tool 04

Antenna Array Factor

Uniform linear array beam pattern. Adjust element count, spacing, and scan angle to shape the main lobe.

AF(θ) = sin(N·ψ/2) / N·sin(ψ/2)
🎯
Tool 05

Smith Chart

Plot normalised load impedance on an interactive Smith Chart. Reads out Γ, VSWR, and return loss live.

Γ = (Z_L − Z₀) / (Z_L + Z₀)
Batch 2 — Systems & Propagation
🛰️
Tool 06

Radar Range Equation

Maximum detection range as a function of Pt, antenna gains, RCS, and receiver sensitivity. Sweep RCS to see range sensitivity.

R⁴ = Pt·Gt·Gr·λ²·σ / (4π)³·L·Smin
📻
Tool 07

Superheterodyne Receiver

Visualise LO, image frequency, and IF architecture. Shows image rejection vs IF bandwidth tradeoff.

f_image = f_RF ± 2·f_IF
📶
Tool 08

FSPL Multi-Band Sweep

Free-space path loss vs distance for up to 4 selectable frequency bands overlaid on one chart.

FSPL = 20·log(4πdf/c)
〰️
Tool 09

Transmission Line

Standing wave pattern |V(z)| along a terminated transmission line, with reflection coefficient on a mini Smith chart inset.

Γ(z) = ΓL · e^(−j2βz)
⚖️
Tool 10

Link Budget Calculator

Editable budget table with running margin, waterfall bar chart, and red highlight when margin goes negative.

Margin = Pr − Sensitivity
