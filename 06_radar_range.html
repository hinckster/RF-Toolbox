Radar Range Equation

Full Skolnik radar range equation — R vs RCS curve, Pr vs range, clutter comparison, pulse compression gain.

Radar Range Equation (Skolnik 2.1)
R⁴ = Pt·Gt·Gr·λ²·σ / [(4π)³·L·S_min]
S_min = kTB·NF·(S/N)_min / T_i
In dB: R_max = ¼ · [Pt+Gt+Gr+2·10log(λ)+σ_dBsm −(4π)³_dB−L−S_min]
λ = c/f. Each term varies as R⁴ — the "fourth-power law".
Received Power
Pr = Pt·Gt·Gr·λ²·σ / [(4π)³·R⁴·L]
= EIRP + Gr + 10log(σ) − 30log(R) + 20log(λ) − const
Falls as R⁻⁴ (40 dB/decade) — every 2× range = −12 dB received power.
Pulse Compression
G_pc = BW·τ = TBP
Range_compressed = Range_uncompressed · TBP^(1/4)
R_max ∝ (Pt·τ·BW)^(1/4) = (E_total)^(1/4)
Time-bandwidth product. LFM chirp: BW = Δf, τ = pulse width.
