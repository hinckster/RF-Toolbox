Transmission Line — Help
What this tool calculates
Computes the complex reflection coefficient ΓL = (ZL − Z₀) / (ZL + Z₀) for a lossless transmission line terminated in a complex load ZL = R + jX. From |Γ|, VSWR = (1 + |Γ|) / (1 − |Γ|) and return loss RL = −20·log₁₀(|Γ|) are derived. The standing wave envelope |V(z)| = |V⁺|·√(1 + |Γ|² + 2|Γ|·cos(2βz + φ)) is plotted along the line, and the input impedance Zin = Z₀·(ZL + jZ₀·tan βL) / (Z₀ + jZL·tan βL) is calculated at the source end.
Sliders & inputs
  • Resistance R (0–300 Ω): Real part of the load impedance. Set to Z₀ (e.g., 50 Ω) for a matched load (|Γ| = 0).
  • Reactance X (−200 to +200 Ω): Imaginary part of the load. Positive values are inductive, negative are capacitive. X = 0 gives a purely resistive load.
  • Z₀ (10–200 Ω): Characteristic impedance of the transmission line. Standard values are 50 Ω (RF/microwave) and 75 Ω (cable TV/video).
  • Line length (0–720°): Electrical length of the line in degrees (0°–720° = 0–2λ). A half-wave (180°) line transforms ZL back to itself; a quarter-wave (90°) inverts it as Z₀²/ZL.
How to read the chart
The main chart shows the normalized voltage magnitude |V(z)|/|V⁺| versus distance from the load (in wavelengths, going left toward the source). The solid cyan line oscillates between VSWR (upper dashed orange) and 1/VSWR (lower dashed orange). A flat line at 1.0 indicates a perfect match. Peaks (voltage maxima) repeat every λ/2. The mini Smith chart inset shows the ΓL point (red dot) and the VSWR circle (dashed copper arc); the load lies on the real axis only when X = 0. SC (short circuit) and OC (open circuit) are labeled on the left and right real-axis edges.
Key results
  • |ΓL|: Reflection coefficient magnitude. 0 = perfect match, 1 = total reflection (short or open).
  • ∠ΓL: Phase angle of Γ. Determines where the first voltage maximum falls on the line.
  • VSWR: Voltage standing wave ratio. 1:1 = perfect match; >2:1 is generally unacceptable for most RF systems.
  • Return Loss (dB): Power reflected back; higher (more positive) is better. >20 dB is excellent.
  • Vmax pos.: Distance from the load to the nearest voltage maximum, in wavelengths.
  • |Zin|: Magnitude of the input impedance seen at the source end of the line.
