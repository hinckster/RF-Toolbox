Transmission Line

Reflection coefficient, VSWR, and standing wave pattern along a lossless transmission line

ΓL = (ZL − Z₀) / (ZL + Z₀)
ZL = R + jX (complex load impedance)
VSWR = (1 + |Γ|) / (1 − |Γ|)
RL = −20·log₁₀(|Γ|) dB
|V(z)| = |V⁺|·√(1+|Γ|²+2|Γ|·cos(2βz+φ))
β = 2π/λ, load at z = 0, source at z = −L
