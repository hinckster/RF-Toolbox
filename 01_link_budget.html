System Link Budget

Full cascaded link budget — Friis equation as the zero-loss special case. Running margin at every stage.

Full Link Budget
Margin = Pr − S_min
Pr = Pt − Ltx + Gt − FSPL − Latm + Gr − Lrx
S_min = kTB + NF + SNRreq + Limpl
Pozar eq. 2.40 — general form. FSPL = 20 log₁₀(4πdf/c)
Friis Equation (simplified: all extra losses = 0)
Pr = Pt + Gt + Gr − FSPL
= Pt + Gt + Gr + 20 log₁₀(λ/4πd)
FSPL [dB] = 20log(dkm) + 20log(fMHz) + 32.44
Friis 1946 — set Ltx=Lrx=Latm=0 to reduce to this form
