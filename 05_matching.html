Smith Chart — Transmission Line Matching

Line length slider rotates Γ around the constant-|Γ| circle in real time. Standing wave pattern updates simultaneously. Single-stub matching solved automatically.

Reflection Coefficient (Pozar 2.4)
Γ_L = (Z_L − Z₀) / (Z_L + Z₀)
z_L = Z_L/Z₀ (normalized impedance)
z = (1+Γ)/(1−Γ) (inverse transform)
VSWR = (1+|Γ|)/(1−|Γ|). Return loss = −20 log|Γ| dB
Transmission Line Rotation
Γ(l) = Γ_L · e^(−j2βl)
β = 2π/λ → 2βl = 4πl/λ
One full rotation = λ/2 line length
Moving toward generator: clockwise rotation. |Γ| stays constant.
Single-Stub Matching
Find l₁: Re(Y(l₁)) = Y₀
Stub length l_s: Im(Y_stub) = −Im(Y(l₁))
Short stub: l_s = (1/2β)·arctan(B/Y₀)
Two solutions exist per half-wavelength. Pozar 5.3.

Smith Chart — Transmission Line Matching — Help
What this tool calculates
Models a load impedance Z_L = R + jX connected to a transmission line of characteristic impedance Z₀. The reflection coefficient at the load is Γ_L = (Z_L − Z₀)/(Z_L + Z₀) (Pozar Eq. 2.4). Adding a length l of lossless transmission line rotates Γ clockwise on the constant-|Γ| circle: Γ(l) = Γ_L · e^(−j4πl/λ). The tool also solves single-stub matching analytically — finding the two line lengths l₁ where Re(Y) = Y₀, then computing the required short- or open-circuit stub length to cancel the residual susceptance.
Sliders & inputs
  • R_L — load resistance in ohms (0–500 Ω). Set to 0 for a short circuit (Γ = −1), very large for open circuit (Γ = +1).
  • jX_L — load reactance in ohms (−300 to +300 Ω). Positive = inductive; negative = capacitive. Moves the Smith Chart point off the real axis.
  • Z₀ — system reference impedance in ohms (1–300 Ω). Typically 50 Ω for RF or 75 Ω for cable TV systems.
  • Line length l/λ — electrical length of the transmission line from 0 to 2λ. Each 0.5λ completes one full clockwise rotation on the Smith Chart. The red dot moves to show Z_in at that line length.
  • Snap presets — buttons that instantly set common load values (50 Ω matched, 75 Ω, short, open, 50+j50, 50−j50) without typing.
How to read the chart
The Smith Chart plots the complex reflection coefficient Γ = Γ_re + jΓ_im. The outer unit circle boundary represents |Γ| = 1 (total reflection). Solid gray circles are constant-resistance (r) contours; the r = 1 circle is highlighted in blue and passes through the center (the matched point). The dashed orange circle is the constant-|Γ| locus — all points on it have the same VSWR as Z_L. The copper dot labeled Z_L is the load; the red dot labeled Z_in is the impedance seen at the generator end of the line. Green and orange filled dots mark the two single-stub matching points. The standing wave chart below shows |V(z)/V⁺| (blue, solid) and |I(z)·Z₀/V⁺| (purple, dashed) vs. distance from load; the red vertical dashed line marks the current line length.
Key results
  • |Γ_L| / ∠Γ_L — magnitude and phase of the load reflection coefficient; fixed regardless of line length.
  • VSWR — voltage standing wave ratio = (1+|Γ|)/(1−|Γ|). Values <1.5 indicate excellent matching; >3 indicates significant mismatch.
  • Return Loss — power reflected back, in dB = −20·log|Γ|. Higher values mean less reflected power.
  • Z_in (R, jX) — input impedance seen looking into the line at the current length l.
  • Single-Stub solutions — two solutions showing l₁ (distance from load to stub junction) and stub length for both short-circuit and open-circuit stubs.
Things to try
  • Snap to 50+j50 Ω and slowly drag the line-length slider — watch the red Z_in dot trace the constant-|Γ| circle clockwise and reach the real axis at the single-stub solution points marked in green/orange.
  • Set R_L = 0 (short circuit) and confirm the Smith Chart dot sits at Γ = −1 (left edge); then sweep line length to watch impedance transform through all purely imaginary values — confirming a shorted stub is a pure reactance element.
  • Load the "50+j50" snap, then find the line length where Z_in ≈ 50+j0 Ω (real axis intersection at the r=1 circle) — this is where a single shunt stub would complete the match.
