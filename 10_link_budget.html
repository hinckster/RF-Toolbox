Link Budget Calculator — Help
What this tool calculates
Performs a cascaded link budget in dB following the Friis transmission equation: Pr = Pt + Gt + Gr − FSPL − Lcables − NF. Free-space path loss is computed as FSPL = 20·log₁₀(4πdf/c) where d is range in metres and f is frequency in Hz. Link margin = Pr − Sensitivity; it represents the dB headroom above the receiver sensitivity floor.
Sliders & inputs
  • Frequency (100 MHz–100 GHz, log scale): Carrier frequency. FSPL grows as 20·log₁₀(f), so doubling frequency adds 6 dB of path loss.
  • Distance (0.1 km–1000 km, log scale): Link range. FSPL grows as 20·log₁₀(d); doubling distance adds 6 dB.
  • Tx Power Pt (dBm): Transmitter output power delivered to the antenna port.
  • Tx Antenna Gain Gt (dBi): Boresight gain of the transmit antenna over an isotropic radiator.
  • Tx Cable/Connector Loss (dB): Ohmic losses between PA output and antenna feed. Enter as a positive number (e.g., 1 for 1 dB loss).
  • Atmospheric Gas Loss (dB): Gaseous absorption and rain fade margin — especially relevant above 10 GHz. Enter as a positive number.
  • Rx Antenna Gain Gr (dBi): Boresight gain of the receive antenna.
  • Rx Cable/Connector Loss (dB): Feed-line loss between receive antenna and LNA input. Enter as a positive number.
  • Noise Figure Margin NF (dB): System noise figure plus implementation loss allowance. Enter as a positive number.
  • Sensitivity (dBm): Minimum detectable signal for the required SNR/BER; displayed in orange as a reference floor.
How to read the chart
The Budget Waterfall chart is a horizontal stacked bar showing how the signal level evolves through each link stage. Green bars are gains (Pt, antenna gains); orange/red bars are losses. Each bar starts where the previous one ended, so the right edge of the last bar is the received power Pr. The vertical dashed orange line marks the sensitivity floor — any bar that crosses to the left of it will be highlighted red in the table above.
Key results
  • Received Power Pr: Total signal power at the receiver input after all gains and losses.
  • Sensitivity: The required minimum power for reliable detection (user-set).
  • Link Margin: Pr − Sensitivity in dB. Green ≥ 6 dB (healthy), orange 0–6 dB (thin), red < 0 dB (link fails).
