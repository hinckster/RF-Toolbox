Tool 03 · Nonlinearity

Two-Tone Intermodulation Test

Polynomial device model (a₁, a₂, a₃) → exact IIP3/OIP3/IIP2/P1dB. Spectrum shows IM2, IM3, and harmonics.

Nonlinear Device Model
y(t) = a₁x + a₂x² + a₃x³
x(t) = A·cos(ω₁t) + A·cos(ω₂t)
a₁ = small-signal gain, a₂ → 2nd-order, a₃ → 3rd-order products
Intercept Points (Razavi)
IIP3 = √(4|a₁| / 3|a₃|) [V pk]
IIP2 = |a₁ / a₂| [V pk]
P1dB: 3|a₃|A²/4 = 0.109|a₁|
At IIP3: fundamental power = IM3 power at output (Pozar 10.34)
Output Spectrum Amplitudes
Fund: a₁A + 9a₃A³/4
IM3 (2f₁−f₂): 3a₃A³/4 [close-in]
IM2 (f₁+f₂): a₂A² [wideband]
2nd harm (2f₁): a₂A²/2
A = V peak per tone into 50 Ω
