Tool 05

Smith Chart

Interactive impedance plot on the complex reflection coefficient plane

Γ = (Z_L − Z₀) / (Z_L + Z₀)
Z_L = R + jX (load impedance)
VSWR = (1 + |Γ|) / (1 − |Γ|)
RL = −20·log₁₀(|Γ|) dB
z = Z_L / Z₀ (normalised)
Constant-r circles: centre (r/r+1, 0), radius 1/(r+1)
Const-x arcs: centre (1, 1/x), radius 1/|x|
All circles pass through Γ = +1 (open circuit)
