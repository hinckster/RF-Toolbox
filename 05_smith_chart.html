Smith Chart — Help
What this tool calculates
Maps any load impedance Z_L = R + jX onto the complex reflection coefficient plane using Γ = (Z_L − Z₀)/(Z_L + Z₀). From Γ it derives VSWR = (1+|Γ|)/(1−|Γ|), return loss = −20·log₁₀|Γ| dB, admittance Y_L = G + jB, Q factor = |X|/R, and mismatch loss = −10·log₁₀(1−|Γ|²) dB. The L-Match Network Synthesis section computes both sign solutions (±Q) of the two-element L-network that transforms Z_L → Z₀, rendering each as a labeled SVG circuit diagram showing the actual component type (inductor or capacitor) for the series and shunt elements.
Sliders & inputs
  • Z₀ — reference (system) impedance in ohms (1–400 Ω). Changes the normalization; all Smith Chart circles rescale relative to this value.
  • Resistance R — real part of load impedance (0–500 Ω). Dragging left toward 0 Ω approaches a short circuit (Γ = −1, left edge of chart); large R approaches open circuit (Γ = +1, right edge).
  • Reactance X — imaginary part of load impedance (−300 to +300 Ω). Positive X = inductive load (upper half of chart); negative X = capacitive (lower half). X = 0 keeps the point on the real axis.
  • Quick Presets — buttons to instantly jump to canonical loads: Matched (50 Ω), Short, Open, 50+j50, 100−j75, 25 Ω, 75 Ω, 100 Ω.
  • Load Scenario — dropdown presets for real-world scenarios including inductive antenna load, capacitive coax-to-antenna, near-matched, and severe open-circuit mismatch.
How to read the chart
The outer circle is the unit |Γ| = 1 boundary — points here represent total reflection (short or open circuits). The horizontal real axis runs from SC (short, left) to OC (open, right); the center is the perfectly matched point z = 1 + j0. Solid gray circles are constant-resistance contours (r = 0.2, 0.5, 1, 2, 5, 10); the r = 1 circle (slightly darker) passes through center. Arcs are constant-reactance contours (x = ±0.2, ±0.5, ±1, ±2, ±5). The three dashed copper circles at |Γ| = 0.25, 0.5, 0.75 are reference VSWR circles. The pink/red dot is Z_L; the copper vector from center shows the Γ phasor; the dashed copper circle through the dot is the constant-VSWR locus for that load.
Key results
  • Z_L / z (norm.) — actual and Z₀-normalized impedance; z = Z_L/Z₀ is what the Smith Chart directly plots.
  • |Γ| / ∠Γ — reflection coefficient magnitude and phase angle; |Γ| = 0 is perfect match, |Γ| = 1 is total reflection.
  • VSWR — standing wave ratio; 1.0 = no standing wave, higher values indicate worse mismatch.
  • Return Loss — power reflected back to source in dB; RL > 20 dB means <1% power reflected.
  • Y_L (G, B) — admittance in mS; useful for shunt matching element design.
  • Q factor — |X|/R; indicates how narrowband a matching network must be. High Q means narrower bandwidth.
  • Mismatch Loss — power lost to reflection, in dB = −10·log₁₀(1−|Γ|²).
  • L-Match Network Synthesis — both ±Q solutions of the L-network that transforms Z_L → Z₀. If R_L ≠ 0, shows Solution A and Solution B as SVG circuit diagrams with component types (L or C) and reactance values. If the load has reactance, a Step 1 series element cancels it before the L-network.
Reference
P. H. Smith, "Transmission line calculator," Electronics, vol. 12, no. 1, pp. 29–31, Jan. 1939. See also D. M. Pozar, Microwave Engineering, 4th ed. Hoboken, NJ: Wiley, 2012, §2.4.
