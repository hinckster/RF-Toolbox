Receiver Dynamic Range

Cascaded noise figure (Friis) + cascaded IIP3 + noise floor + SFDR — the complete receiver DR analysis in one view.

Friis NF Cascade (Pozar 10.1)
F_sys = F₁ + (F₂−1)/G₁ + (F₃−1)/(G₁G₂) + ···
NF_sys = 10 log₁₀(F_sys) [dB]
F = 10^(NF/10). First stage dominates if G₁ is large.
IIP3 Cascade (Pozar 10.3)
1/IIP3_sys² = 1/IIP3₁² + G₁²/IIP3₂² + ···
OIP3 = IIP3 + Gain [dB, per stage]
Last high-gain stage usually limits IIP3. OIP3_sys = IIP3_sys + G_total.
Dynamic Range
SFDR = ⅔(OIP3_sys − N_floor) [dB]
N_floor = kTB + NF_sys [dBm]
P1dB ≈ OIP3 − 9.6 dB [rule of thumb]
SFDR: largest signal before IM3 = noise. Spurious-free dynamic range.

Receiver Dynamic Range — Help
What this tool calculates
Performs a complete cascaded receiver chain analysis using Friis's noise figure formula and the cascaded IIP3 formula. Computes system noise floor (kTB + NF), output P1dB (OIP3 − 9.6 dB rule), and Spurious-Free Dynamic Range: SFDR = ⅔(OIP3_sys − N_floor). All four stages are editable inline.
Inputs (inline table)
  • NF (dB): Noise Figure of each stage. LNA NF is most critical — even 0.5 dB increase ripples through the whole cascade.
  • Gain (dB): Stage gain (positive) or insertion loss (negative, e.g., filter = −1 dB). High first-stage gain suppresses downstream NF but degrades IIP3.
  • IIP3 (dBm): Input-referred third-order intercept. Mixers and amplifiers at high gain typically limit system IIP3.
  • System Bandwidth: Sets noise floor (kTBF). 20 MHz ≈ 5G NR, 200 kHz ≈ LTE narrowband.
  • Input signal level: Reference signal power — shows where your signal sits within the dynamic range window.
How to read the charts
Dynamic Range Window (left): Horizontal bars show: noise floor (bottom), your input signal, IM3 products, P1dB (compression onset), and OIP3. SFDR is the gap between the noise floor and where IM3 products reach it.

Stage-by-Stage buildup (right): Cascaded NF (blue) rises with each stage — ideally flat after a high-gain LNA. Cascaded OIP3 (orange) typically drops at high-gain stages.
Key results
Cascaded NF — total system noise figure; IIP3/OIP3 — input/output intercept referred to system input/output; SFDR — maximum dynamic range before IM3 products rise above noise; P1dB — estimated compression point. Red/orange/green color coding indicates poor/marginal/good performance.
Things to try
  • Reduce LNA NF from 1.5 to 0.8 dB — watch the system NF drop nearly 1:1, demonstrating first-stage dominance (Friis).
  • Increase LNA Gain from 20 to 30 dB — IIP3 degrades as the mixer now sees higher input power; observe the SFDR tradeoff.
  • Set BW = 200 kHz (LTE NB-IoT) vs 100 MHz (5G mmWave) — noise floor shifts 27 dB; see how SFDR changes.
Reference
Pozar, Microwave Engineering, 4th ed., Wiley, 2012, §10.1–10.3 (Friis cascade, IIP3 cascade, SFDR).
