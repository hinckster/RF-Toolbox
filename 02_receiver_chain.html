Receiver Dynamic Range

Cascaded noise figure (Friis) + cascaded IIP3 + noise floor + SFDR — the complete receiver DR analysis in one view.

Friis NF Cascade (Pozar 10.1)
F_sys = F₁ + (F₂−1)/G₁ + (F₃−1)/(G₁G₂) + ···
NF_sys = 10 log₁₀(F_sys) [dB]
F = 10^(NF/10). First stage dominates if G₁ is large.
IIP3 Cascade (Pozar 10.3)
1/IIP3_sys² = 1/IIP3₁² + G₁²/IIP3₂² + ···
OIP3 = IIP3 + Gain [dB, per stage]
Last high-gain stage usually limits IIP3. OIP3_sys = IIP3_sys + G_total.
Dynamic Range
SFDR = ⅔(OIP3_sys − N_floor) [dB]
N_floor = kTB + NF_sys [dBm]
P1dB ≈ OIP3 − 9.6 dB [rule of thumb]
SFDR: largest signal before IM3 = noise. Spurious-free dynamic range.
