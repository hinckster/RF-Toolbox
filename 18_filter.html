  • Resonator: Set R = 1 Ω, L = 10 nH, C = 10 pF → Q ≈ 200. Increase R to 50 Ω and observe BW widen to ~100 MHz. Compare Series vs Parallel to see dual Q behavior.

Filter Design & Resonance — Help
Resonator mode
Computes the resonant frequency f₀ = 1/(2π√LC), Q factor, and 3 dB bandwidth for a series or parallel RLC resonator. The impedance |Zin|(ω) is plotted over a ±2 decade sweep around f₀.

For a series resonator: |Z| is minimum (= R) at f₀; Q = ω₀L/R = 1/(ω₀RC). High Q → narrow BW, sharp series resonance. Used as notch filters or trap circuits.

For a parallel resonator: |Z| is maximum (≈ R) at f₀; Q = R/(ω₀L) = Rω₀C. High Q → high impedance peak. Used as bandpass tank circuits in oscillators and filters.
LC Filter mode
  • Butterworth: Maximally flat magnitude response in passband. −3 dB exactly at fc. Rolloff = −20n dB/decade beyond cutoff.
  • Chebyshev: Equal-ripple passband. Steeper rolloff than Butterworth for the same order at the cost of in-band ripple. Passband is 0–fc.
  • Low-pass topology: Shunt capacitor → series inductor → shunt capacitor … (shunt-C first ladder). Alternate topology reverses the element order.
  • High-pass topology: LP→HP frequency transformation replaces each shunt C with a shunt L, and each series L with a series C.
Component values are denormalized to the target cutoff frequency fc and reference impedance Z₀. The frequency response is computed exactly using ABCD (T-matrix) cascade, accounting for source and load Z₀.
Prototype g-values
The normalized LP prototype values g₁…gn are computed analytically. For Butterworth: gk = 2·sin[(2k−1)π/2n]. For Chebyshev with ε dB ripple: the g-values follow from the closed-form recurrence involving β = ln[coth(ε·ln10/40)] and γ = sinh(β/2n).
