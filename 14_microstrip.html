Microstrip / Stripline Calculator
What this tool calculates
Computes the characteristic impedance Z₀, effective dielectric constant ε_eff, phase velocity vp, guided wavelength λ_g, and quarter-wave length for microstrip and centered-stripline geometries using closed-form expressions. Synthesis mode inverts the equations to find the required trace width W for a target Z₀ given substrate parameters.
Formulas used
Microstrip — Hammerstad–Jensen (1980). Accurate to <0.5% for 0.05 ≤ W/H ≤ 20. Conductor thickness T is not included in the quasi-static Z₀; for T/H > 0.05, the effective width is reduced by ΔW ≈ T/π · ln(2H/T).

Stripline — Cohn's formula for a centered conductor between parallel ground planes. B is the total board thickness. T = 0 is protected against division by zero internally.
Inputs
ε_r — substrate relative permittivity at RF. FR4 ≈ 4.0–4.8 (lossy above 2 GHz); Rogers RO4003C = 3.55; RT/duroid 5880 = 2.2 (low-loss); alumina = 9.9.
H — substrate height (microstrip) or total board thickness B (stripline), in mm.
T — conductor thickness in mm. Standard values: 1 oz Cu = 0.035 mm; 2 oz Cu = 0.070 mm.
W — trace width in mm. The Analysis section computes Z₀ from W; Synthesis inverts this.
f — frequency in GHz. Z₀ and ε_eff are quasi-static (frequency-independent); f is used only to compute λ_g and λ/4.
Chart
Shows Z₀ vs W/H on a log x-axis. Four reference curves (air, RT5880, FR4, alumina) plus a highlighted curve for the current ε_r. The copper dot marks the current operating point. Use this to visually compare substrates and read off W/H for a target impedance.
Things to try
  • Load FR4, set Target Z₀ = 50 Ω in the synthesis section — then drag W to match the result and confirm the Z₀ card reads 50 Ω.
  • Switch to RT/duroid 5880 (ε_r = 2.2) — observe Z₀ rises for the same W/H. Lower permittivity requires a wider trace to stay at 50 Ω.
  • Enable Stripline mode — Z₀ drops compared to microstrip for the same W/B because the conductor is fully enclosed in dielectric (no air above).
  • Set f = 24 GHz (24 GHz automotive radar) on RO4003C — check λ/4 to size a quarter-wave matching stub or bandpass filter resonator.
  • Use Export CSV to save the full Z₀ vs W/H sweep for post-processing in a spreadsheet or script.
