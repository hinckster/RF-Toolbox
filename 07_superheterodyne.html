Tool 07

Superheterodyne Receiver

LO planning, image frequency, and image rejection vs IF bandwidth

f_LO = f_RF + f_IF  (high-side injection)
f_LO = f_RF − f_IF  (low-side injection)
LO mixes with RF to down-convert to IF; injection side sets image location
f_image = f_RF + 2·f_IF  (high-side)
f_image = f_RF − 2·f_IF  (low-side)
Image is 2·f_IF away; preselector BPF must reject it
IRR ≈ 20·log₁₀(f_image / BW_IF)
Conceptual rejection model — wider IF BW = harder image rejection
