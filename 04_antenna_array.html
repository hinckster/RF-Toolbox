ULA Array Factor

Element pattern × amplitude taper → total array pattern. Compare uniform vs tapered; grating lobe markers; Balanis Ch. 6.

Array Factor (Balanis 6-10)
AF(θ) = Σ aₙ · e^(jnψ)
ψ = kd·sinθ + β [k = 2π/λ, θ from broadside]
Uniform: |AF| = sin(Nψ/2) / sin(ψ/2)
aₙ = amplitude taper weights, β = progressive phase shift for beam steering
Element Pattern × Array Factor
E_total(θ) = E_elem(θ) × AF(θ)
Isotropic: E_elem = 1
Cosine: E_elem = |cosθ|
Patch: E_elem = cos²θ
Pattern multiplication principle — valid for identical, uniformly spaced elements
Grating Lobes
θ_GL = arcsin(sinθ₀ ± m·λ/d)
Occur when d/λ > 1/(1+|sinθ_steer|)
HPBW ≈ 0.886·λ/(N·d·cosθ₀) [rad]
Grating lobes appear for d ≥ λ. Sidelobe level varies with taper window.

ULA Array Factor — Help
What this tool calculates
Computes the normalized far-field pattern of a Uniform Linear Array (ULA) using the Array Factor formula AF(θ) = Σ aₙ · e^(jnψ), where ψ = kd·sinθ + β (Balanis Eq. 6-10). The total radiated pattern is the product of the element pattern (isotropic, cosine, or patch cos²θ) and the AF — the pattern multiplication principle. Directivity is approximated as D ≈ 2Nd/λ for broadside ULAs with isotropic elements.
Sliders & inputs
  • Elements N — number of array elements (2–64). More elements narrow the main beam and increase directivity by ~3 dB per doubling.
  • Spacing d/λ — element spacing normalized to wavelength (0.1λ–2.0λ). Half-wavelength (0.5λ) is the standard; exceeding 1/(1+|sinθ₀|) causes grating lobes.
  • Steer angle θ₀ — progressive phase shift β = −kd·sinθ₀ steers the main beam from −60° to +60° off broadside.
  • Element Pattern — isotropic (unity), cosine |cosθ|, or patch cos²θ. Non-isotropic patterns suppress end-fire lobes via pattern multiplication.
  • Amplitude Taper (Array A) — windowing function applied to element weights: Uniform (−13 dB SLL, narrowest beam), Hamming (−43 dB SLL), Hann (−31 dB SLL), Chebyshev equiripple (−40 dB SLL), or Taylor −35 dB.
  • Array B controls — independent element count, spacing, and taper for the dashed-purple comparison overlay.
How to read the chart
The Cartesian view plots normalized pattern magnitude in dB (−60 to 0 dB) vs. angle θ from −90° to +90°. The solid blue line is Array A; the dashed purple line is Array B. Vertical dashed red lines mark grating lobe positions where d/λ causes additional main-beam-strength lobes at arcsin(sinθ₀ ± mλ/d). The Polar tab renders the same data on a 360° radar chart with 0 dB at the rim and −60 dB at center.
Key results
  • Directivity — approximated as 10·log₁₀(2Nd/λ) dBi, valid for broadside ULA with isotropic elements.
  • HPBW — half-power beamwidth (−3 dB points) of the main lobe in degrees; narrows as N·d grows.
  • 1st SLL — peak sidelobe level in dBc relative to main beam peak; determined by taper window.
  • Grating Lobes — lists angles (degrees) where full-strength grating lobes appear, or "None" if d/λ is small enough to prevent them.
Things to try
  • Set N=16, d/λ=0.5, steer to 30° — watch grating lobes appear at the red dashed lines as you increase d/λ past 0.87λ (the grating-lobe threshold at 30° scan).
  • Compare Array A (Uniform) vs. Array B (Hamming) with N=32 — observe the 30 dB SLL reduction at the cost of ~1.5× beam broadening.
  • Switch element pattern from Isotropic to Patch (cos²θ) with large steer angle — note how the element pattern suppresses sidelobes near ±90° that the AF would otherwise show.
Reference
C. A. Balanis, Antenna Theory: Analysis and Design, 4th ed. Hoboken, NJ: Wiley, 2016, Ch. 6 (Arrays) and Ch. 7 (Planar Arrays).
