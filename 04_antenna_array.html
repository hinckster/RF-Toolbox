Tool 04

Antenna Array Factor (ULA)

Uniform Linear Array beam pattern with phase-steering

AF(θ) = sin(N·ψ/2) / [N · sin(ψ/2)]
normalised array factor for N isotropic elements
ψ = k·d·cos(θ) + β
k = 2π/λ  |  β = −k·d·cos(θ₀)
d = element spacing, θ₀ = scan angle, θ = observation angle
HPBW ≈ 0.886·λ / (N·d) [rad]
Directivity ≈ N·d/λ · 2 (broadside, λ/2 spacing)
first null at ψ = ±2π/N
